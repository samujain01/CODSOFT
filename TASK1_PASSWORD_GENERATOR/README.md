# Password Generator

## Description
A simple Python application that generates strong and random passwords based on the length entered by the user.

## Features
- User-defined password length
- Random password generation
- Includes letters, numbers, and special characters

## Technologies Used
- Python

## How to Run
1. Open the project folder.
2. Run:
   python password_generator.py
3. Enter the desired password length.
4. The generated password will be displayed.
